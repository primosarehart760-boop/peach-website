## 🔀 3 个交汇点（中间可视化 · 核心）

### 交汇 ① · PRD 功能 → AI 训练师判"要不要 AI"
- PM 的 PRD Lite 列出 10 个功能 → AI 训练师分类：
  - 🟢 强 AI 场景（必须 AI · 如智能分诊 / 智能信贷）
  - 🟡 AI 可选（AI 辅助 · 传统也能做 · 如智能客服）
  - 🔴 非 AI 场景（传统工程 · 不上 AI · 如登录 / 权限）
- 可视化：左→右箭头 · 分三色标签贴到 AI 训练师模型选型卡上

### 交汇 ② · AI 方案 → 反要求 PRD 加"反指标 / 边界异常 / 验收"
- AI 训练师定了模型 + 评测集 → 反向要求 PM 的 PRD 加：
  - 幻觉率 ≤ X% 作为反指标
  - 模型挂时的边界异常处理（降级兜底）
  - Gherkin 验收标准（Given 用户问 X / When 模型幻觉 / Then 兜底回复）
- 可视化：右→左箭头 · 红色高亮贴到 PM 的 PRD Lite 和指标体系卡上

### 交汇 ③ · 行业专家 → 两边灌"合规红线 + 行业 KPI"
- 从顶部下发 · 同时流向左右两侧：
  - 合规红线（金融：等保三级 · 医疗：HIPAA · 教育：未保）
  - 行业 KPI（金融：误拒率 · 医疗：分诊准确率 · 电商：POR）
- 可视化：顶→下双箭头 · 贴到 PM 指标体系 + AI 训练师评测集

## 🎭 前端视角 · 实现

### 核心技术点
- **双栏布局**（grid-template-columns: 1fr auto 1fr · 中间是交汇点窄列）
- **双向箭头动画**（SVG path + CSS animate · 或者 CSS 的 pseudo element）
- **文档卡**（同 step 风格 · 可展开）
- **行业专家下发**（顶部横条 · 从顶往下流）
- **协作日志**（可滚动时间线 · 最新在上）

### 数据驱动
- 接 URL 参数（同 workspace）· 读需求 + 行业 + 模式
- 根据行业加载对应协作数据
- 闪电档：快速渲染 · 5+5 文档 1 分钟全出 + 交汇点静态显示
- 标准档：慢节奏 · 文档逐份生成 · 交汇点实时触发

### 箭头动画
- 交汇 ①：左 → 右 · 绿/黄/红三色 pill 从 PRD 功能飘到 AI 选型
- 交汇 ②：右 → 左 · 红色 pill 从 AI 方案飘到 PRD + 指标
- 交汇 ③：上 → 下双分叉 · 橙色 pill 从行业专家飘到左右两侧
