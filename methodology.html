01AI 训练师 10 大 SOP

这不是"懂点 AI 术语"· 是从模型选型到备案上线的全链路方法论。每个 SOP 都有精确数字和可执行动作 · 不是 PPT 空话。

🎯 模型选型

5 维决策矩阵（合规 / 中文 / 场景 / 成本 / 延迟）· 3 链路（主 / 备用 / 降级）· 场景映射 6 类任务。

Qwen Max 主豆包 Pro 备DeepSeek V3 降

📏 评测集设计

500 条起步 · 5 大类 × 每类 100 · 3 分布（40 通过 / 30 边界 / 30 拒绝）· 合规 100% 硬门槛。

专业情感穿帮合规记忆

🔬 评测方法

LLM-as-Judge（自动周度）+ 人工精标（周 50 条）+ Promptfoo A/B（上线前必跑）· 不同模型自评避偏袒。

穿帮率 < 0.5%幻觉率 < 5%

✍️ Prompt 工程

三层结构（Meta / System / User）+ 6 层设计（Role / Context / Task / Constraints / Examples / Output）+ 反引导话术库。

Few-shot禁词拦截反破防

🔧 模型调优

Badcase 飞轮（周度）+ SFT 微调（10k 数据）+ DPO 偏好对齐（1k pair）+ LoRA 行业专属（500 案例训练 2 周）。

A100 × 1月度迭代

🔁 数据飞轮

5 环（用户对话 → 每日摘要 → 人工标注 → 训练数据 → 上线验证）· 越用越聪明 · 竞品追赶成本指数级。

M1 · 85%M12 · 95%

🛡 降级策略

主链挂 → 备用 → 降级 → 兜底话术 · 切换 < 30 秒 · 健康检查每 10 秒 · 用户无感。

错误率 > 5%延迟 > 10s

🚨 内容安全

3 层防御（正则 10ms + AI 盾 300ms + 人工抽检 10%）· 心理危机硬编码拦截 · Prompt 注入 3 类防御。

涉黄 100%涉政 100%危机 100%

📋 算法备案

Day 1 启动（3 月流程）· 6 份必备材料 · 工信部审核 · 备案号公示规范。

T-90 启动代理 ¥30k

🧠 RAG 知识库

分层（4-6 库）+ bge-m3 嵌入 + Milvus 向量 + Rerank 精排 · 命中率 > 85% · P95 < 3s。

chunk 500 字重叠 50
