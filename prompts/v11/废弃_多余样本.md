## 二、产品定位

### 一句话定位公式

> **为 K12 学生家长（孩子 6-18 岁）** 提供 **AI 个性化错题讲解 + 家长周报协同能力** · 解决 **"家长没时间辅导 + 自己讲不明白 + 家教 600/月太贵"** 痛点 · 区别于猿辅导的关键差异是 **"AI 反代做检出 + 家长/老师/学生三方周报协同"**。

- **目标用户**：林小雨 · 35 岁 · 北京浦东 · 市场部经理 · 孩子读小学四年级 · 教育预算 2000 元/月
- **核心问题（JTBD）**：我在晚上 8 点孩子写作业求问的场景 · 想要快速帮孩子搞懂错题 · 因为自己累了一天不想讲 + 讲不明白 · 替代方案"猿辅导"要预约 + 贵 600/月 · 让我觉得不值。
- **核心价值主张**：家长不用开口 · AI 讲清错题 + 每周自动汇报进度
- **与竞品差异化 3 条**：
  1. **家长周报协同**（猿辅导无）· 每周五晚 8 点自动发家长
  2. **AI 代做检出**（其他 AI 辅导都没有）· 防止孩子让 AI 代做作业
  3. **价格**：69 元/月（比猿辅导 600/月便宜 88%）

## 四、使用场景

### 场景 1：课后作业批改（高频主流程）

- **用户角色**：林小雨（家长）+ 林同学（四年级学生）
- **触发动机（JTBD）**：晚上 8 点 · 林同学数学作业有 3 道题不会 · 找妈妈讲 · 妈妈加班累不想讲
- **操作步骤**：
  1. 林同学拍照上传错题（1 步）
  2. AI 分析 → 给出讲解 + 3 道类似题（2 步）
  3. 林同学标记"懂了" · 妈妈周五看周报
- **达成效果**：3 道错题 15 分钟内搞懂 · 妈妈不用讲 · 周报自动推送进度
- **为什么选小桃学习 vs 猿辅导**：不用预约 / 24h 可用 / 每月省 531 元
- **验收标准**：3 道错题 AI 讲解用时 ≤ 2 min · 标"懂了"比例 ≥ 70%

### 场景 2：周五家长周报（中频）

- **用户角色**：林小雨
- **触发动机**：每周五晚 8 点 · 想知道孩子本周学习状况 · 但没时间翻所有作业
- **操作步骤**：收到 App 推送 → 打开周报 → 3 屏看完（整体 / 薄弱点 / 下周建议）
- **达成效果**：3 分钟了解孩子进度 + 下周重点
- **验收标准**：周报阅读 3 分钟内完成 · 打开率 ≥ 80%

### 场景 3：AI 代做检出告警（低频 · 关键）

- **用户角色**：系统自动 + 林小雨 + 老师
- **触发动机**：林同学 10 分钟内问同一道题 5 次 · 系统怀疑在用 AI 代做
- **操作步骤**：系统检测 → 推送家长 + 老师 → 家长介入
- **达成效果**：防止孩子依赖 AI · 回归自主学习
- **验收标准**：检出后家长 24h 内响应率 ≥ 90%

### 5.2 功能详细说明（以"AI 错题讲解"为例）

#### 功能 1：AI 错题讲解

- **功能描述**：学生拍照错题 → OCR + AI 结合课纲 + 学生历史数据 → 个性化讲解 + 3 道类似题
- **前置条件**：已登录 + 已付费（或体验次数 > 0）+ 上传清晰错题照片
- **操作流程**：
  1. 首页点"拍照批改"大按钮
  2. 拍照 / 相册选图
  3. 系统 OCR 识别题干 + 学生答案
  4. 调 Claude Sonnet 4.6 + 课纲 RAG（BGE-m3 向量库）
  5. 展示讲解（3 屏：原题 → 错在哪 → 正确解法）+ 3 类似题
  6. 学生标"懂了" / "还不懂" → 写入学习数据

- **业务规则**：单日上限 20 题（防滥用）· 图片 ≤ 5MB · 支持科目：数学 / 语文 / 英语 / 物理
- **AI 边界**（N34）：**必须 AI**（规则无法穷举讲解逻辑 · 需语义理解 + 个性化）· 推荐 Claude Sonnet 4.6（教育场景温度 0.4）· 合规 C 端可用（已脱敏 + 告知家长）
